# WHITE ROOM Phase 14 Demo

This walkthrough is for the local chat-first cockpit.

1. Open `http://127.0.0.1:8765/`.
2. Confirm the home route lands in the chat cockpit.
3. Review the top bar for the active lane, runner health, and usage summary.
4. Use the composer to inspect local, manual, and gated provider lanes.
5. Open the right rail to review project memory, route decision, and manual lanes.
6. Resize the browser to about 380px wide and confirm the layout still reads as a cockpit, not a dashboard.

The demo stays local-first and does not call cloud providers automatically.
