# WHITE ROOM Phase 14B Demo

This walkthrough shows the quiet-first local cockpit experience.

1. Open `http://127.0.0.1:8765/`.
2. Confirm the home redirect lands in chat for the active project.
3. Open `http://127.0.0.1:8765/chat/white-room`.
4. Check that the sidebar starts compact, with projects, conversations, and the compact Tools menu only.
5. Open the Tools menu and confirm the seven operational destinations are reachable.
6. Open the context drawer only when needed.
7. Switch lane or mode in the composer and reload the page to confirm the last choice is restored.
8. Expand the sidebar, reload, and confirm the disclosure state persists.
9. Use the command palette for quick local actions when you want to jump without leaving the cockpit.

The demo is complete when the chat surface remains primary, the drawer stays hidden by default, and the quiet layout returns on reload.
