# Notice

WHITE ROOM is an original local-first AI workbench concept focused on reducing AI usage cost, preserving project memory, routing work across model lanes, and keeping privacy/security controls visible.

Original project direction:

- frequent AI users need a structured way to build production-grade work
- high-capability models should be reserved for planning, hard reasoning, and review
- cheaper, free, or local models should handle scoped execution and repetitive work
- project memory should stay local and reusable
- provider calls should be visible, approval-gated, and cost-aware

This notice is included to preserve attribution for the project concept and public positioning.
